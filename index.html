Settings features

This is used to depict the overall look and feel of the shop settings in it the various features like security, data clearance, themes and data ranges are selected here.

  1. Color themes
    1. Change between various themes Light or Dark themes
  2. Security Settings
    1. Enable Biometric/password security feature for the app

      2. By enabling security one has to use Biometrics i.e (Facial recognition,fingerprint option or the password) to access the application

      By disabling this one can access the application without security.

  3. Data retrival Range
    1. 3 Day Range (retrieved from server)
    2. 7 Day Range (retrieved from server)
    3. Monthly Range (retrieved from server)
    4. All data (is retrieved)
  4. Local data Clearance
    1. Kisenyi shop Data clearance

Part of the hidden entities of the settings are the Errors management systems.

This are part and Essential for providing feedback to the users when some of the operations in the App are done.
